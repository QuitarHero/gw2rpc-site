# gw2bot-site

> The website of [GW2RPC by Maselkov](https://github.com/Maselkov/GW2RPC).

> [gw2rpc.info](https://gw2rpc.info/)

## Build Setup

This project uses [Flask](http://flask.pocoo.org/).

``` bash
# Install Flask, requires Python
pip install Flask

# Run it at localhost:5000
py rpcsite.py
```
